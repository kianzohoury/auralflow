Weighted L2 loss using 2 or 3 components depending on arguments.

Balances the target source separation quality versus the amount of residual noise attenuation. Optional third component balances the quality of the residual noise against the other two terms.

auralflow.losses.losses.kl_div_loss(mu: torch.FloatTensor, sigma: torch.FloatTensor) → torch.Tensor#

Computes KL term using the closed form expression.

KL term is defined as := D_KL(P||Q), where P is the modeled distribution, and Q is a standard normal N(0, 1). The term is combined with the reconstruction loss.
