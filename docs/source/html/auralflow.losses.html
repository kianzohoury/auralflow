class auralflow.losses.KLDivergenceLoss(model, loss_fn: str = 'l1')[source]#

Bases: torch.nn.modules.module.Module

Wrapper class for kl_div_loss.

The KL loss term is defined as \(D_KL(P||Q)\), where \(P\) is the modeled distribution, and \(Q\) is a standard normal distribution \(N(0, 1)\). In closed form, the loss can be expressed as:

\[D_{KL}(P||Q) = \frac{1}{2} \sum_{i=1}^{n}(\mu^2 + \sigma^2 - \ln(\sigma^2) - 1)\]
where:

  • \(\mu\): mean of the modeled distribution \(P\)

  • \(\sigma\): standard deviation of modeled distribution \(P\)

  • \(n\): number of tensor elements

Note

Sets batch_loss attribute of the model instead of returning the loss directly. Can also only be used with SpectrogramNetVAE.

Parameters

  • model (SpectrogramMaskModel) -- Spectrogram mask model.

  • loss_fn (str) -- Construction loss criterion. Default: 'l1'.

forward() → None[source]#

Calculates construction loss + KL loss and stores it.

class auralflow.losses.SIDRLoss(model)[source]#

Bases: torch.nn.modules.module.Module

Wrapper class for si_sdr_loss.

Typically used as an evaluation metric, but can be optimized for directly. For a single audio track, the loss is defined as:

\[L_{\text{SI-SDR}}(\hat y, y) = -10\log_{10} \frac{||\frac{proj_{y} \hat y}{||y||_2^2}||_2^2}{||\frac{proj_{y} \hat y}{||y||_2^2} - \hat y||_2^2}\]
where:

  • \(y\): true target signal

  • \(\hat y\): estimated target source signal

Note

auralflow.losses.component_loss(mask: torch.FloatTensor, target: torch.FloatTensor, residual: torch.FloatTensor, alpha: float = 0.2, beta: float = 0.8) → torch.Tensor[source]#

Weighted loss that combines different loss components.

Uses two or three loss components depending on the arguments for alpha and beta. If beta=0, it weighs the quality of target source separation against the amount of residual noise attenuation. If beta>0, the third component balances the quality of the attenuated residual noise against the other two terms. The 2-component loss is defined as:

\[L_{2c}(X; Y_{k}; \theta; \alpha) = \frac{1-\alpha}{n} ||Y_{f, k} - |Y_{k}|||_2^{2} + \frac{\alpha}{n}||R_f||_2^{2}\]

and the 3-component loss is defined as:

\[L_{3c}(X; Y_{k}; \theta; \alpha; \beta) = \frac{1-\alpha -\beta}{n} ||Y_{f, k} - |Y_{k}|||_2^{2} + \frac{\alpha}{n}||R_f||_2^{2} + \frac{\beta}{n}|| \hat{R_f} - \hat{R}||_2^2\]
where:

  • \(Y_{f, k} := M_{\theta}\odot |Y_{k}|\), filtered target

  • \(R_{f} := M_{ \theta }\odot (|X| - |Y_{k}|)\), filtered residual

  • \(\hat{R_{f}} := \frac{R_{f}}{||R_{f}||_2}\), filtered unit-residual

  • \(\hat{R} := \frac{R}{||R||_2}\), unit residual

Parameters

  • mask (FloatTensor) -- Estimated soft-mask (output of the network).

  • target (FloatTensor) -- Ground-truth target source.

  • residual (FloatTensor) -- The residual or background noise, defined as the difference between the mixture and target sources.

  • alpha (float) -- Value for the two-term component loss.

  • beta (float) -- Value for the three-term component loss.

Returns

Mean component loss.

Return type

Tensor
