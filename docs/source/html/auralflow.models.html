class auralflow.models.SeparationModel(config: dict)[source]#

Bases: abc.ABC

Interface shared among all source separation models.

Should not be instantiated directly but rather subclassed. A subclass must implement the following methods: set_data, forward, compute_loss, backward, optimizer_step and separate.

Variables

  • target_labels (List[str]) -- Target source labels.

  • criterion (Union[nn.Module, Callable]) -- Loss function.

  • optimizer (Optimizer) -- Optimizer.

  • scheduler (Any) -- LR scheduler.

  • batch_loss (FloatTensor) -- Current batch loss.

  • train_losses (List[float]) -- Epoch training loss history.

  • val_losses (List[float]) -- Epoch validation loss history.

  • stop_patience (int) -- Waiting time in epochs before reducing LR, if validation loss does not improve.

  • max_lr_steps (int) -- Max number of LR reductions before stopping early.

  • use_amp (bool) -- If True, uses automatic mixed precision.

  • grad_scaler (Any) -- Gradient scaler (only if using automatic mixed precision).

  • is_best (bool) -- Flag for checkpointing the best model.

  • model_params (dict) -- Model configuration parameters.

  • visualizer_params (dict) -- Visualizer configuration parameters.

  • training_mode (dict) -- If True, model is in training mode; otherwise model is in inference mode.

  • device (str) -- Device.

class auralflow.models.SpectrogramNetSimple(num_fft_bins: int, num_frames: int, num_channels: int = 1, hidden_channels: int = 16, mask_act_fn: str = 'sigmoid', leak_factor: float = 0, dropout_p: float = 0.5, normalize_input: bool = False, normalize_output: bool = False, device: Optional[str] = None)[source]#

Bases: torch.nn.modules.module.Module

Vanilla spectrogram-based deep mask estimation model.

Parameters

  • num_fft_bins (int) -- Number of FFT bins (aka filterbanks).

  • num_frames (int) -- Number of temporal features (time axis).

  • num_channels (int) -- 1 for mono, 2 for stereo. Default: 1.

  • hidden_channels (int) -- Number of initial output channels. Default: 16.

  • mask_act_fn (str) -- Final activation layer that creates the multiplicative soft-mask. Default: 'sigmoid'.

  • leak_factor (float) -- Alpha constant if using Leaky ReLU activation. Default: 0.

  • dropout_p (float) -- Dropout probability. Default: 0.5.

  • normalize_input (bool) -- Whether to learn input normalization parameters. Default: False.

  • normalize_output (bool) -- Whether to learn output normalization parameters. Default: False.

  • device (optional[str]) -- Device. Default: None.

class auralflow.models.SpectrogramNetVAE(*args, **kwargs)[source]#

Bases: auralflow.models.architectures.SpectrogramNetLSTM

Spectrogram U-Net model with a VAE and LSTM bottleneck.

Encoder => VAE => LSTM x 3 => decoder. Models a Gaussian conditional distribution p(z|x) to sample latent variable z ~ p(z|x), to feed into decoder to generate x' ~ p(x|z).
