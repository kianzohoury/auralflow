auralflow.models.SeparationModel#

class auralflow.models.SeparationModel(config: dict)[source]#

Interface shared among all source separation models.

Should not be instantiated directly but rather subclassed. A subclass must implement the following methods: set_data, forward, compute_loss, backward, optimizer_step, scheduler_step and separate.
