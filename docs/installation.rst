Installation
============

.. toctree::

System Requirements
-------------------

Auralflow currently requires Python version >= 3.7.11, <3.11.

Installation
------------

Install auralflow via PyPi package manager:

.. code-block:: console

   $ pip install auralflow

Updating Auralflow
------------------

Update auralflow to the latest version:

.. code-block:: console

   $ pip install <package_name> --upgrade
